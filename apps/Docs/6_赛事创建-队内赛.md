# 队内赛创建
## 名词解释
| 术语 | 说明 |
| --- | --- |
| 球局 | 普通创建或快速创建的普通球局，可以是一组或多组人，不带球队ID |
| 赛事/队内赛 | 专指球队比赛，带有球队ID的球局 |
| 组（Group） | **物理概念**，一个Tee Time一起打球的人，通常2-4人 |
| 队/分队（TAG） | **逻辑概念**，队内赛中临时分成的阵营，由赛事创建者手工添加两个及以上TAG（如：东邪队、西毒队、南帝队、北丐队），用于统计对抗结果 |
| 旺波 | 即 OneBall 的音译，中文标准翻译是"最佳球位"或"四人四球最佳球位" |


---

## 赛制说明
### 大类划分
+ **比杆赛**：按总杆数计算胜负，支持多个分队同时比赛
+ **比洞赛**：按每洞胜负计算，赢洞多者获胜，**仅限两个分队对抗**

### 具体赛制
| 赛制 | 分队要求 | 分组规则 | 记分方式 |
| --- | --- | --- | --- |
| 个人比杆赛 | 可选 | 随意安排 | 个人成绩 |
| 四人四球最好成绩比杆赛 | 必须 | 每组须含两队成员（1v1/1v2/2v1/2v2） | 团队成绩 |
| 四人四球最佳球位比杆赛（旺波比杆） | 必须 | 每组须含两队成员（1v1/1v2/2v1/2v2） | 团队成绩 |
| 四人两球比杆赛 | 必须 | 每组必须2v2（4人） | 团队成绩 |
| 个人比洞赛 | 可选 | 每组2人，1v1 | 个人成绩 |
| 四人四球最好成绩比洞赛 | 必须 | 每组须含两队成员（1v1/1v2/2v1/2v2） | 团队成绩 |
| 四人四球最佳球位比洞赛（旺波比洞） | 必须 | 每组须含两队成员（1v1/1v2/2v1/2v2） | 团队成绩 |
| 四人两球比洞赛 | 必须 | 每组必须2v2（4人） | 团队成绩 |


### 人数说明
+ **个人比洞赛**：每组只安排2人，进行1v1对决
+ **四人四球**：每组2-4人，须包含两个分队的成员，支持1v1、1v2、2v1、2v2（双方人数无需相等）
+ **四人两球**：每组必须4人，必须2v2

### 记分说明
+ **个人赛制**：每人单独记分
+ **团队赛制**（四人四球/四人两球）：只记录团队成绩，系统不参与过程记录
    - 四人四球最好成绩 vs 四人四球最佳球位（旺波）在系统记分上展现一致，仅赛制表述不同

### 比洞赛结果展示
比洞赛结果使用专业格式展示：

**比赛进行中（胜负未分）：**

| 格式 | 含义 |
| --- | --- |
| XUP | 计分卡**上方**一方领先X洞 |
| XDN | 计分卡**下方**一方领先X洞 |
| AS | All Square，当前平局 |


**比赛结束（胜负已分）：**

| 格式 | 含义 | 示例 |
| --- | --- | --- |
| X&Y | 领先X洞，剩余Y洞时提前锁定胜局 | "3&2" 表示领先3洞，还剩2洞时比赛结束 |
| XUP | 打完全部洞后，计分卡上方一方领先X洞获胜 | "1UP" 表示上方领先1洞 |
| XDN | 打完全部洞后，计分卡下方一方领先X洞获胜 | "1DN" 表示下方领先1洞 |
| AS | All Square，打完全部洞后平局 | - |

## 分组与分队
### 分组（Group）
物理概念，指一个 Tee Time 一起打球的人。

**分组规则：**

+ 个人比杆赛：分组可随意安排
+ 个人比洞赛：每组只安排2人，进行1v1
+ 四人四球（比杆/比洞）：每组须包含两个分队的成员，支持1v1、1v2、2v1、2v2
+ 四人两球（比杆/比洞）：每组必须4人，必须2v2

### 分队（Sub-team）
逻辑概念，用于统计对抗结果。

**分队规则：**

+ 由赛事创建者手工添加分队，可自定义名称（如：东邪队、西毒队、南帝队、北丐队）
+ **比杆赛**：可设置两个及以上分队
+ **比洞赛**：仅限两个分队对抗
+ 个人赛制：分队可选
+ 四人四球/四人两球：必须分队

## 创建队内赛
### 创建入口
从球队页面进入创建队内赛

### 创建字段
| 字段 | 类型 | 说明 |
| --- | --- | --- |
| 球队名称 | 选择 | 选择所属球队（用户可能是多个球队成员） |
| 比赛名称 | 文本输入 | 自定义比赛名称 |
| 比赛场地 | 选择 | 选择球场 |
| 比赛时间 | 日期时间 | 开球时间 |
| 参赛费用 | 数字输入 | 仅展示信息，不涉及支付 |
| 比赛赛制 | 选择 | 8种赛制选一 |
| 奖项设置 | 文本输入 | 纯文本描述 |
| 分组权限 | 单选 | 见下方说明 |
| 分队设置 | 配置 | 仅团队赛制时必填 |


### 分组权限
控制谁来决定球员分配到哪个组（Tee Time）：

+ **管理员分组**：由管理员/创建者指定球员分配到各组
+ **球员自由选择分组**：球员报名后自己选择加入哪个组

### 分队设置
仅当选择四人四球或四人两球赛制时需要配置。

+ 赛事创建者手工添加分队，可自定义名称
+ **比杆赛**：可设置两个及以上分队（如：东邪队、西毒队、南帝队、北丐队）
+ **比洞赛**：仅限两个分队
+ 球员报名后需选择加入某一分队

### 取前N名成绩
+ 团队赛制时，每个分队取前N名成绩进行汇总排名
+ **默认值**：如果创建时未输入N值，系统以**最少参赛人数分队的人数**作为默认N值
  - 示例：东邪队报名6人，西毒队报名8人 → 默认N=6

---

## 报名规则
+ **报名限制**：球队外用户报名需要批准,文案提示报名者:"可能被驳回/拒绝"
+ **赌球**：不在队内赛层面考虑，赌球与具体"球局"（组）相关，由选手自行决定

---

## 赛事状态流转
```plain
创建 → 报名中 → 报名截止 → 进行中 → 已结束
                 ↓
               已取消
```

| 状态 | 说明 |
| --- | --- |
| 报名中 | 球队成员可报名参赛 |
| 报名截止 | 停止报名，管理员进行分组 |
| 进行中 | 比赛开始，可以记分 |
| 已结束 | 比赛完成，展示最终结果 |
| 已取消 | 比赛取消 |
