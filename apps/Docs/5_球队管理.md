## 球队
创建者自动为超级管理员

## 球队基本信息
+ 球队名称
+ 球队logo
+ 球队口号(slogan)

## 约定
+ 一个球队必须有且只能有一个超级管理员
+ 整个系统内,球队名称不能重复
+ 一个用户可以属于多个球队

## 权限集合:
+ 审批入队
+ 拉入队员
+ 踢出队员
+ 对某场比赛的某个玩家标注已经付费
+ 创建队内赛
+ 财务统计

## 管理员类别
超级管理员可添加其他管理员，普通管理员的权限范围由超级管理员赋予。

+ SuperAdmin/超级管理员
+ admin/普通管理员



## 球队角色
+ SuperAdmin 超级管理员 ,只能有一个
+ admin 普通管理员,可以有多个
+ member 成员 ,可以有多个



## 超级管理员特殊场景
### 超级管理员自行转让
超级管理员可以将"超级管理员"身份转让队内任意一人

### 超级管理员退出球队
当球队超级管理员退出球队，应由系统自动指派超级管理员权限给其他队员

指派方法: 可从权限范围最大的队员中选择,如果出现多个候选人,则指定加入球队
