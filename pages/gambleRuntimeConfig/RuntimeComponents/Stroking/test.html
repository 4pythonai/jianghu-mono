Stroking 让杆配置组件

组件功能

  • ✅ 用户选择：支持从游戏玩家中选择让杆用户
  • ✅ PAR值配置：支持PAR3、PAR4、PAR5的让杆数值设置
  • ✅ 洞范围选择：可选择起始洞和结束洞
  • ✅ 数据管理：支持多用户配置的保存和更新
  • ✅ 响应式设计：适配不同屏幕尺寸

使用方法

                    // 1. 在页面JSON中引入组件
{
  "usingComponents": {
    "Stroking": "/pages/gambleRuntimeConfig/RuntimeComponents/Stroking/Stroking"
  }
}

// 2. 在WXML中使用
<Stroking 
  strokingConfig="{{strokingConfig}}"
  bind:save="onStrokingSave"
  bind:cancel="onStrokingCancel"
/>

// 3. 在JS中处理事件
Page({
  data: {
    strokingConfig: []
  },
  
  onStrokingSave(e) {
    const { config } = e.detail;
    this.setData({ strokingConfig: config });
  },
  
  onStrokingCancel() {
    console.log('取消配置');
  }
});

配置数据格式

                    [
  {
    "userid": "张三",
    "holeRanges": [7, 8, 9, 10, 11],
    "PAR3": 1,
    "PAR4": 0.5,
    "PAR5": 0.5
  },
  {
    "userid": "李四", 
    "holeRanges": [10, 11, 12, 13],
    "PAR3": 1,
    "PAR4": 0.5,
    "PAR5": 0.5
  }
]

组件特性

让杆数值范围： -1, -0.5, 0, 0.5, 1
支持负数： 可以为水平较高的用户设置负让杆
多用户支持： 可以为多个用户分别配置让杆
洞范围灵活： 支持任意洞范围的让杆配置

注意事项

  • 组件依赖 gameStore 中的玩家数据和洞数据
  • 确保在使用前已正确初始化游戏数据
  • 让杆配置会影响计分计算，请谨慎使用
  • 建议在游戏开始前完成让杆配置
